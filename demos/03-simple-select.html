RowEntry DataTable Plugin

Simple Select Demo

This demo shows using RowEntry to provide a select list of items that can be added to the table.

Notice that items that are in the table are disabled in the select list. When they are deleted from the table, they are re-enabled in the select list.

  • Select an item from the select list, and then click the green 'add' icon to add the item to the table
  • Click the red 'delete' icon next to a row to delete the row
