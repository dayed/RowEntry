RowEntry

RowEntry is a plug-in (meta-plug-in ?) for the excellent DataTables jQuery plugin.
It provides a very flexible interface for adding rows to the table and deleting rows from the table.

The plugin provides an easy-to-use wrapper around the DataTables plugin, but you are responsible for providing the html element to be presented to the user to specify a new row, and for providing the data for the new row when the user chooses to add it.

Here are some examples to get you started:

  • Simple text field
  • Text field, with a radio button column in the table
  • A Select List providing entry options
The Git repo for this project is at https://github.com/rfreedman/RowEntry
